Expect

class Expect : public object

断言期望对象

Expect 对象为 assert 模块内部对象，用于实现 expect 风格断言，创建：

var test = require("test");
var obj = test.expect(exp);

或者直接引用：

var expect = require("expect");
var obj = expect(exp);

或者通过 test.setup 配置：

require("test").setup();
var obj = expect(exp);
