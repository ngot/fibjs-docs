assert

断言测试模块，如果测试值为假，则报错，报错行为可设定继续运行或者错误抛出

引用方法：

var assert = require('assert');

或者通过 test 模块引用：

var test = require('test');
var assert = test.assert;

或者通过 test.setup 配置：

require("test").setup();
