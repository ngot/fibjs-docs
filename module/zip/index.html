open

ZipFile open(String path,String mod,Integer compress_type)

打开一个zip文件

Parameters

  • path 文件路径

  • mod 打开文件模式, “r”代表读取, “w”代表创建, “a”代表在zip文件后追加

  • compress_type 压缩类型, ZIP_STORED 代表不压缩, 仅存储。 默认使用ZIP_DEFLATED 代表使用zlib库进行压缩。

Returns

返回zip文件对象
