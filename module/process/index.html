memoryUsage

Object memoryUsage()

查询当前进程内存使用报告

Returns

返回包含内存报告

内存报告生成类似以下结果：

{
  "rss": 8622080,
  "heapTotal": 4083456,
  "heapUsed": 1621800
}

其中：

  • rss 返回进程当前占用物理内存大小

  • heapTotal 返回 v8 引擎堆内存大小

  • heapUsed 返回 v8 引擎正在使用堆内存大小

open

SubProcess open(String command,Array args,Object opts)

运行指定的命令行，接管进程输入输出流，并返回进程对象

Parameters

  • command 指定运行的命令行

  • args 指定运行的参数列表

  • opts 指定运行的选项，支持的选项如下：

    {
"timeout": 100, // 单位为 ms
"envs": [] // 进程环境变量
}

Returns

返回包含运行结果的进程对象
