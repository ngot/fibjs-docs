memoryUsage

Object memoryUsage()

查询当前进程内存使用报告

Returns

返回包含内存报告

内存报告生成类似以下结果：

{
  "rss": 8622080,
  "heapTotal": 4083456,
  "heapUsed": 1621800,
  "nativeObjects": 122
}

其中：

  • rss 返回进程当前占用物理内存大小

  • heapTotal 返回 v8 引擎堆内存大小

  • heapUsed 返回 v8 引擎正在使用堆内存大小

  • nativeObjects 返回当前有效内置对象数
